# Backend API

This is the backend for the MERN Stack application.

## Prerequisites

- Node.js installed
- MongoDB Atlas account or local MongoDB

## Setup

1.  Navigate to the `backend` directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Create a `.env` file in the `backend` root based on `backend/.env.example`.
4.  Start the server:
    ```bash
    npm run dev
    ```
